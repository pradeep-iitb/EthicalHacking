# Intro to Reconnaissance 

Reconnaissance is the initial phase of a penetration test or security assessment when hacker gathers as much information as possible about target system or network .

Reconnaissance can be **Passive** *(Without direct interaction , public sources , lower risks , less detailed and limited info ,less resource intensive )* or **Active** *( Direct interaction , port scanning , probing , higher risk , more accurate info , more resource and toll intensive )*

## Google dorking

Google hacking refers to use of advanced search operators in Google to find specific info , not visible through regular search queries .

***Some Google Dorks***
* site:  --  Restricts search results to a specific domain  eg. site:microsoft.com
* filetype:  --  Searches for specific file types eg. filetype:pdf
* inurl:  --  Finds urls containing a specific string eg. inurl:login
* intitle:  --  looks for pages with specific words in title eg. intitle:Google dorking
* allintitle:  --  Looks for pages with all specified title 
* intext:  --  Searches for specific text with body of pages

## WHOIS Records & DNS

WHOIS Records provide detailed info about the registered owner of a domain name including contact detailes , registration dates , status and maintained by domain registrars in WHOIS Databases . Various online sites povides these records as well as some commands also provides this info 

``` bash
whois microsoft.com
```

DNS translates alphabetic domain names to the numeric IP addr of computers and servers . 

| Record Type | Description |
| :--- | :--- |
| **A** | Maps a domain to an IPv4 address. |
| **AAAA** | Maps a domain to an IPv6 address. |
| **CNAME** | Aliases one domain name to another. |
| **NS** | Specifies the authoritative nameserver for a domain. |
| **SOA** | Contains administrative information about the domain, like the primary nameserver and zone update settings. |
| **TXT** | Stores text information, often used for verification. |
| **MX** | Directs email to mail servers. |

Below commands can be useds for DNS Recon

``` bash
host microsoft.com // returns IP addr of th host 
host -t ns microsoft.com // returns the name-sever of host
nslookup microsoft.com
>> set type=ns
>> microsoft.com
```

Subdomain is a part of a larger doamin name . It helps organize and navigate differnet sections or areas of a website . Subdomain enumeration means finding all available subdomains with sites and kali tools -


## Identifying WAF 

Web Appilication Firewall is a security system designed to monitor and control the incoming and outgoing network traffic based on predetermined security rules . It can be identified with wafw00f

``` bash

```
